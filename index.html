Tentang Saya

Halo!! Saya Resti Fauzi Mukhliza, seorang mahasiswa di STMIK Mardira Indonesia yang memiliki minat besar dalam pengembangan web dan desain UI/UX. Saya aktif berorganisasi dan berfokus pada pengembangan sistem dan aplikasi yang interaktif serta ramah pengguna. Pengalaman proyek saya mencakup pembuatan Sistem Informasi Perpustakaan, Chatbot untuk Konsultasi Kesehatan, dan Sistem Inventory Berbasis Web untuk sebuah toko kosmetik. Melalui proyek-proyek ini, saya telah mengasah keterampilan dalam merancang dan menerapkan solusi yang efisien, dan saya berkomitmen untuk terus mengembangkan keahlian saya di bidang ini.

Proyek

ChatBot Perobatan Konsultasi Kesehatan

Chatbot Perobatan Konsultasi Kesehatan adalah sebuah aplikasi kelompok berbasis AI yang dirancang untuk memberikan informasi dan saran terkait kesehatan, terutama mengenai penggunaan obat-obatan. Sebagai Data Scientist, saya bertanggung jawab untuk menganalisis dan memproses data yang digunakan dalam pengembangan Chatbot Perobatan Konsultasi Kesehatan. Tugas saya meliputi pengumpulan dan pembersihan data medis, serta mengembangkan model analisis untuk memahami pola dan tren penggunaan obat.

Sistem Informasi Perpustakaan

Sistem Informasi Perpustakaan adalah aplikasi kelompok berbasis web yang dirancang untuk mempermudah pengelolaan dan akses informasi perpustakaan. Sebagai desainer antarmuka, saya bertanggung jawab untuk menciptakan desain sistem antarmuka yang intuitif dan menarik untuk Sistem Informasi Perpustakaan. Tugas saya mencakup merancang tata letak, dan membuat elemen navigasi yang memudahkan pengguna dalam mengakses berbagai fitur, seperti pencarian buku, peminjaman, dan pengembalian.

Sistem Inventory Berbasis Web untuk sebuah toko kosmetik

Sistem Inventory Berbasis Web adalah aplikasi kelompok yang dirancang khusus untuk mengelola stok produk di Toko Ethernal Charm Cosmetic. Proyek ini bertujuan untuk meningkatkan efisiensi operasional dan mempermudah pemantauan serta pengelolaan inventaris produk kosmetik. Sebagai analisis sistem, tugas saya meliputi analisis kebutuhan pengguna dan pengumpulan informasi untuk mendefinisikan fitur dan fungsionalitas yang diperlukan dalam sistem.
